# 📚 Library Management System in C

A simple **Library Management System** developed in **C** using fundamental **Data Structures** such as Linked Lists, Stacks, and Queues. This project demonstrates how data structures can be used to efficiently manage library operations.

---

## 📖 Features

- ➕ Add a new book
- 🔍 Search for a book by ID
- 📚 Display all books
- ✏️ Update book details
- ❌ Delete a book
- 🔃 Sort books by Book ID
- 📌 View recently added books using a Stack
- 📥 Borrow books using a Queue
- 📤 Return borrowed books
- ✅ Duplicate Book ID validation

---

## 🛠️ Data Structures Used

### Linked List
Stores all books in the library dynamically.

### Stack
Maintains a record of recently added books.

### Queue
Handles the borrowing and returning of books in First-In-First-Out (FIFO) order.
